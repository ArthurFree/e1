# 决策记录

| 决策 | 结论 | 原因 |
| --- | --- | --- |
| 项目位置 | 仓库根目录的 `notion-like-web` | 用户要求与现有桌面项目完全独立 |
| 首版形态 | 完整 Web 笔记应用 | 不只复刻单页编辑器，还包括知识库管理 |
| 数据保存 | IndexedDB 本地优先 | 无账号、离线可用、后续可迁移云端 |
| 参考标准 | 1440 × 900 严格视觉回归 | 以指定 Notion-like 模板的布局和交互为目标 |
| 本地化 | 简体中文 | 面向中文使用；视觉对比基于中文基线 |
| Tiptap 许可 | 使用开源等价实现 | 避免将首版绑定到 Pro 许可 |
| AI | 用户配置的 OpenAI-compatible provider | 兼容多种服务且不需要应用方托管密钥 |
| 首版排除 | 协作、账号、云同步、评论与权限 | 优先交付可用的单机 Web 笔记体验 |
| Markdown 导入导出 | Tiptap 官方 `@tiptap/markdown` 扩展 | 官方开源方案，解析经编辑器扩展白名单，避免引入 marked/turndown 双库 |
| R001 知识库结构（v0.2） | 全局开始首页 + 知识库首页双层结构；`folder` 迁移为 `group` | 对齐语雀信息架构；分组语义比分组浏览器式文件夹更明确，迁移保留 ID 与层级 |
| 代码高亮 | `lowlight` 离线语法集（common） | 无运行时网络请求，满足本地优先；语言保存在代码块属性，未知语言回退纯文本 |
| 版本历史 | 5 分钟间隔自动版本 + 内容去重 + 自动版本每文档上限 100 | 控制 IndexedDB 膨胀；恢复前强制存 `before-restore` 版本避免二次丢失 |

任何会改变以上结论的需求，应先更新本文件及受影响的需求、架构和测试文档。
